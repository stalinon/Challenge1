# Задача
Есть класс, который занимается обработкой файлов разного формата (Html, Text, JSON …).
Предполагается, что будут добавляться новые форматы файлов. Напишите реализацию,
позволяющую расширять форматы обрабатываемых файлов. Детали обработки конкретного
содержимого файла неважны. Дополнительно, покройте реализацию тестами.

    public class FileProcessor
    { 
        public void ProcessFile(string fileName)
        {
             
        }  
 
    }
  
#
* Для добавления нового формата файла придется написать лишь один класс с всего одним методом, 
обрабатывающим содержимое файла, и свойством, содержащим формат.
* Обработка содержимого файла следующая - файл считывается в строку, в конец через пробел добавляется слово,
строка записывается в файл.
* Тесты реализованы с XUnit - запуск функции, проверка, действительно ли в конце файла искомое слово.
